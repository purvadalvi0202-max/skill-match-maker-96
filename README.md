Smart Resume Matcher 🚀

AI-Powered Resume Screening & ATS Scoring Platform

📌 Project Overview

Smart Resume Matcher is an AI-powered web application that automates resume screening and ATS scoring for recruiters. The system parses resumes, extracts candidate information, calculates ATS compatibility scores, and generates AI-based recruiter feedback.

The platform helps recruiters save time by automatically ranking candidates based on job descriptions and skill matching.
